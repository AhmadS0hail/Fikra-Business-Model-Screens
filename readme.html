Fikra-Business-Model-Screens In order to setup this project on your machine simply do the following: 1- Clone the project (https://github.com/akmail2/Fikra-Business-Model-Screens.git) 2- import database file from DB folder 3- change database name , user and password in file wp-config.php 
        
        (define( 'DB_NAME', 'dbname' );

        /** Database username */
        define( 'DB_USER', 'username' );

        /** Database password */
        define( 'DB_PASSWORD', 'pass' );)
4- open wp-content/themes/fikra. 5- run : npm run build Thant's all 👍🏻
